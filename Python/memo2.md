## 分散とは
「データがどれくらいバラついているか」を数値で表したもの  
分散の定義は、平均からの差の二乗の平均  

### 例
テストの点数がそれぞれ (50,60,70,70,100) の分散を求める場合  
 - 平均を計算
 (50+60+70+70+100)/5 = 70  
 
 - 平均からの差の二乗を計算  
 (50 - 70)^2 = 400  
 (60 - 70)^2 = 100  
 (70 - 70)^2 = 0  
 (70 - 70)^2 = 0  
 (100 - 70)^2 = 900  
 
- 計算結果の平均を計算
(400+100+0+0+900)/5=280  

答えは、280
